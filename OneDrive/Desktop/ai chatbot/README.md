# AI Student Chatbot

This project implements an AI chatbot using Natural Language Processing.

## Technologies:
- Python
- NLP
- TF-IDF
- Cosine Similarity
- Scikit-learn

## Features:
- Answers student queries
- Knowledge-based chatbot
- NLP similarity matching

## Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the chatbot:

```bash
python chatbot.py
```

## Example Conversation

```
AI Chatbot: Ask me anything (type 'exit' to stop)

You: what is python
Chatbot: Python is a high level programming language used for web development, data science, and AI.

You: what is AI
Chatbot: Artificial Intelligence is the simulation of human intelligence in machines.

You: exit
Chatbot: Goodbye!
```
